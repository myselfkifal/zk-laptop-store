# Zk. — Premium Laptop E-Commerce Experience

**Zk.** is a high-end, dark-themed digital showroom for elite laptops. Built with a focus on modern aesthetics and fluid animations.

![Zk Store Preview](https://raw.githubusercontent.com/myselfkifal/zk-laptop-store/main/public/preview.png)

## Features

- **Premium Glassmorphic UI**: Powered by **Tailwind CSS v4** for a sleek, translucent look.
- **Fluid Animations**: Integrated **Framer Motion** for hero entrances, card hover effects, and page transitions.
- **Flying Cart Interaction**: Custom micro-interaction where items glide to the cart icon.
- **Full E-Commerce Flow**: Includes a dedicated shop, cart management (persistence via LocalStorage), and checkout.
- **Modern Tech Stack**: React 19 + Vite + Tailwind v4 + Lucide Icons.

## Getting Started

### Prerequisites

- Node.js (v18+)
- npm or yarn

### Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/myselfkifal/zk-laptop-store.git
   cd zk-laptop-store
   ```

2. Install dependencies:

   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

## Built With

- **React** - Component-based architecture
- **Vite** - Lightning-fast build tool
- **Tailwind CSS v4** - Next-gen utility-first styling
- **Framer Motion** - High-fidelity animations
- **Lucide React** - Modern iconography
